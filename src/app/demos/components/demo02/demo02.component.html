Démo 02 - Les pipes

Les pipes sont des fonctions qui transforment une valeur affichée dans le template HTML

Les pipes sont utilisées dans les expressions d'interpolation {{ expression | nomPipe}}

Angular fournit des pipes intégrés : nombres, dates, devises, pourcentages et chaines de caractères

Exemples de pipes intégrés

String

UpperCasePipe: {{ "voici ma chaine de caractères" | uppercase }}

LowerCasePipe: {{ "VOICI UNE CHAINE DE CARACTÈRES" | lowercase }}

TitleCasePipe: {{ "voici ma chaine de caractères" | titlecase }}

Number (DecimalPipe)

{{ valeur | number [ : 'format' [ : locale ] ] }}

Base: {{ 123456.789 }}

Base: {{ 123456.789 | number }}

Base: {{ 123456.789 | number : '8.4-5' }}

Base: {{ 123456.789 | number : '4.1-2' }}

Base: {{ 123456.789 | number : '4.1-2' : 'fr-BE' }}

Base: {{ 123456.789 | number : '4.1-2' : 'nl-BE' }}

Date (DatePipe)

Base: {{ now }}

Base: {{ now | date }}

Base: {{ now | date : 'd MMMM yyyy HH:mm:ss' : '+0100' }}

Base: {{ birthdate | date : 'd MMMM yyyy HH:mm:ss' }}

Base: {{ birthdate | date : 'd MMMM yyyy HH:mm:ss' : '+0100' }}

Base: {{ now | date : 'd MMMM yyyy HH:mm:ss' : '+0100' : 'en-US' }}

Base: {{ now | date : 'd MMMM yyyy HH:mm:ss' : '+0100' : 'fr-BE' }}

Base: {{ now | date : 'd MMMM yyyy HH:mm:ss' : '+0100' : 'nl-BE' }}

Les pipes personnalisés

Les pipes personnalisés sont des classes qui implémentent l'interface PipeTransform

Commande pour créer un pipe: ng g p nom_pipe

Exemple de pipe pour capitaliser des mots (sans paramètres)

Valeur de départ: {{ "bonjour tout le monde !" }}

Valeur de départ: {{ "bonjour tout le monde !" | capitalize }}

Exemple de pipe pour tronquer un texte (avec un paramètre)

Valeur de départ: {{ "Voici une chaine de caractères avec une certaine longueur." }}

Valeur de départ: {{ "Voici une chaine de caractères avec une certaine longueur." | truncate }}

Valeur de départ: {{ "Quentin Geerts" | truncate }}

Valeur de départ: {{ "Voici une chaine de caractères avec une certaine longueur." | truncate : 10 }}

Exemple de pipe pour masquer une partie de chaines (avec plusieurs paramètres)

Valeur de départ: {{ "1234567890" }}

Valeur de départ: {{ "1234567890" | mask : 3 : "•" : 'start' }}

Valeur de départ: {{ "1234567890" | mask : 4 : "*" : 'end' }}
